# semaphore-api

## Tested in

- Python 3.10.11

## Clone the project

```
git clone https://github.com/isnandar1471/semaphore-api.git
```

## Install dependensi

```
pip install -r requirements.txt
```

## Run app

```
uvicorn src.app:app --reload --host 0.0.0.0 --port 8000
```

python main.py
